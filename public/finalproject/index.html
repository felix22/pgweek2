Now that the header is all styled up, we're going to style the body, that contains all of our content.

The body is divided into two sections (body-top and body-bottom), the first one having a white background and the bottom part having a dark grey background.





Part 2 – Body-Top

Step 1 – Style the Container

We'll start styling the first section of the body, by adding a div to the page under the header, with a class of body-top


      <div class="body-top">

      </div>

In the css, we'll style this body-top content container setting its width to 960px, centering it with margin 0 auto and adding some padding to the left and right sides like so:

        
        .body-top{

        width: 960px;
        padding: 0 10px;
        margin: 0 auto;
        }

Step 2 - Style the headers

In this page, we only use the header files twice, but in more complex pages, we use them abit more often so its good practice to style all your header files together, so you dont have to style them individually as needed.

Firstly, we'll target all the header files, and set the font and color we want all of them to have.


        h1, h2, h3, h4, h5, h6 {

        font-family: 'Oswald', sans-serif, Arial;
        color: #222222;
        }

Next, we'll overide the default font-sizes provided by the browser by specifying specific font sizes for all the header tags:


        h1 {font-size: 2.75em; font-weight: bold;}
        h2 {font-size: 2.15em; font-weight: bold;}
        h3 {font-size: 1.85em; font-weight: bold;}
        h4 {font-size: 1.6em;}
        h5 {font-size: 1.4em;}
        h6 {font-size: 1.2em;}

The H1-H3 tags are bolded to give them more emphasis than the H4-H6 tags.

Step 3 - The Content

We'll first add the content of body-top with their appropriate tags and classes. H1 for the header, P for the subhead and labels, and then the button tag for the button.

We'll add a subhead class to the first P tag, a “label” class to the second so we can style them differently and we'll target the button with a .btn class, so we can use the code again later. We'll style the H1 and Button tags by targeting them with css heirachy. 


        <div class="body-top">

          <h1>Get Things Done on time... Everyday!</h1>
          <p>5things helps you get the "important" things done by limiting your to-do list to 5 items daily, forcing you to focus only on the most important tasks.</p>

          <p>Sign-up in 60 seconds for only $19.99/Year</p>
          <button >Get Started</button>

        </div>

Step 3 - Styling the Content

We'll start by centering the H1 tag in the css with text align center


        .body-top h1{

          text-align: center;
          }

For the sub-head, we'll target it with the “subhead” class and set the font to 'Montserrat', font size to 24px, line-height to 35px so its spaced out properly, center it with text-align center and change the color to a lighter shade of grey


        .subhead{ 

          font-family: 'Montserrat',serif; 
          font-size: 24px; 
          line-height: 35px; 
          text-align: center; 
          color: #545757; 
          }

For the label, we'll target it with a “label” class, set the font to “Montserrat”, its font-size to 16px, remove all paddings and margins so it sits tightly on top of the button, center it with text align center, and change the color to red.


        .label{ 

          font-family: 'Montserrat',serif; 
          font-size: 16px; 
          padding: 0; 
          margin: 0; 
          text-align: center; 
          color: #c0392b; 
          }

I always have problems centering BUTTON tags with margin 0 auto or text align center, because it always just chooses not to work right. So I wrap my buttons in a div with a width set to the width I want my button to be, set the button to width 100% (allowing it to take the full width of the div around it) and then I center the parent div with margin 0 auto.


          <div class="btn-parent"><button class="btn" >Get Started</button></div>
          


        .btn-parent{ 

          width: 420px; 
          margin: 0 auto; 
            }

        .btn{
 
          width: 100%; 
            }

Now we can add more style to .but, setting its background color, borders, font family, font size and its :active and :hover states.


        .btn{ 

            width: 100%; 
            background-color: #fff; 
            color: #c0392b; 
            border: solid #c0392b 2px; 
            border-radius: 4px; 
            font-family: Arial, sans-serif; 
            font-weight: 400px; 
            font-size: 24px; 
            padding: 15px 0; 
        } 

        .btn:hover{ 

            color: #fff; 
            background-color: #c0392b; 

        } 

        .btn:active{ 

            color: #fff; 
            background-color: #c0392b; 
            box-shadow: inset 0 0 3px #fff; 
        }

Finally, we'll place the image directly below the button in body-top,


          <img src="img/5things1.png">

and set its width to 100% so it can center itself properly and never “flow” out of its parent container. We'll also add a bit of padding on the top of the image, to space it out from the button.

 

          .body-top img{ 
       
            width: 100%; 
            padding-top: 23px; 
          }

The next thing we want to do is style the bottom part of the page (body-bottom) and we'll start by adding a div below the body-top div container and giving it a class “ body-bottom”:


          <div class="body-bottom" > 
    

          </div>

In the css, we'll set the background color and give it some padding at the top and bottom.


        .body-bottom { 

          background-color: #2e3639; 
          padding: 30px 0; 

        }

If you look closely at the page we have now, there is a little bit of whitespace between the big image in body-top and the body-bottom container.

Despite my best efforts with paddings and margins, I still couldnt remove it. I did abit of research and found that the space is there due to how images are displayed in the browser by default and this space can be removed by setting the “vertical-align” property to “middle”.

Update your .body-top img selector with "vertical align middle" like this:


         .body-top img{ 
     
            width: 100%; 
            padding-top: 23px; 
            vertical-align: middle; 

        }

Refresh your browser and you'll see that the space is gone.

Now lets start adding content to body-bottom.

The first thing we're going to do is add the “how it works!” header in a h2 tag, and then center it in css,, change its font size and its color to white so its visible on the dark background.


            <h2>How it Works!</h2>
        


          .body-bottom h2 { 

              text-align: center; 
              font-size: 5.9em; 
              color: #fff; 
              padding: 0; 
              margin: 0;

          }

Next, we'll add a div for the white buttom container under the h2, and give it a class of “bottom-container”:

 

        <div class="bottom-container" > 

  
        </div>

Now, lets add some basic style before we add the content. We'll center it with Magin 0 Auto, change the background color to white, and add some padding and margin.

Again, we want the box to have a width of about 620px, a padding of 50px inside on either side and about 30px top and bottom.

Beause of the box model, we'll set the width to be 520px and the padding of 50px on each side will add up to make the total 620px.


        .bottom-container{ 

          margin: 0 auto; 
          background-color: #fff; 
          padding: 30px 50px; 
          width: 520px;

        }

Now, lets add the content of body bottom in their appropriate tags, so that we can begin to style them.


        <div class="bottom-container" > 
          <p>At the start of your day:</p> 
            <p> 
              <ul> 
                <li> Add the 5 most-important things you need to do to 5Things</li> 
                <li> Start with #1 </li> 
                <li> And don't stop until you're done (except for breaks)</li> 
              </ul> 
            </p> 
                       
            <p>Do this and you'll never have an unproductive d...ther system.</p> 

            <p>Start using 5Things to maximise your ...nd get things done today.</p> 

            <!-- BUY BUTTON --> 
            <p class="label">Sign-up in 60 seconds for only $19.99/Year</p> 
            <div class="btn-parent"><button class="btn" >Get Started</button></div> 

            <!-- BUY BUTTON --> 
        
        </div>

For all the P (paragraph) tags in this div, we also need to set the font family to source sans pro, set the font size to 20px and the line height to 1.6. A line height of 1.6 means 1.6 x the font size. So in this case, a line height of 1.6 gives the fonts a line height of 32px. I also set a margin bottom of 23px to space out each paragraph properly.

Every decition here, from the font size to the line height, to even my font choice has been made delibratly to maximise the readability of the content area (i.e. to make it easy to read).


        .bottom-container p { 

            font-family: 'Source Sans Pro', Arial, sans-serif !important; 
            color: #222222;
            font-size: 20px; 
            line-height: 1.6; 
            margin-bottom: 23px; 
            text-rendering: optimizeLegibility; 

        }

We also need to apply this styling (except for the margin bottom) to the Li tags:


        .bottom-container li { 

            font-family: 'Source Sans Pro', Arial, sans-serif !important; 
            color: #222222;
            font-size: 20px; 
            line-height: 1.6; 
            text-rendering: optimizeLegibility; 
        }

Finally, to go back and set the margins and font sizes of the .label class to important, so that the .label styling wont be overridden by the styling for the p tag that contains it.


        .label{ 

            font-family: 'Montserrat',serif ; 
            font-size: 16px !important; 
            padding: 0; 
            margin: 0 !important; 
            text-align: center; 
            color: #c0392b !important; 

        }

The last thing we need to do is add the border, border radius and border shadow styling to the bottom-container, and our page is now fully styled.


        .bottom-container{ 

            margin: 0 auto; 
            padding: 30px 50px; 
            width: 520px; 
            background-color: #fff;  
            border: 1px solid #d0d5cb; 
            border-radius: 5px; 
            box-shadow: 0 0 0 4px rgba(208,213,203,0.3); 

        }
